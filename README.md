# 뱀파이어 키우기 (VampireRaise)

Shrimprium의 플랫폼 방치형 파트를 체리피킹해 만든 웨이브 디펜스형 방치 게임 MVP.
뱀파이어(빨간새우)를 키우고, 몰려오는 인간(파란새우) 웨이브를 막고, 물어서 노예(검은새우)로 부린다.

## 실행

```bash
npx serve .        # http://localhost:3000 접속
npm test           # vitest 단위 테스트
```

빌드 없는 순수 ES 모듈 정적 웹. (file:// 로 직접 열면 모듈이 막히므로 로컬 서버 필수)

## MVP 기능

- **플랫폼/로직 블록**: Shrimprium 블록 59종 재활용 (수질 관련 블록 제외).
  스위치·레드스톤·논리게이트·피스톤·타이머·리피터 신호 시스템 그대로 동작,
  스프링·가시·컨베이어·미끄럼·블랙홀 워프·스턴·투명 게이트 기믹 포함.
- **꾸미기 모드**: Shrimprium과 동일 규칙 — 20px 그리드, 상/하 7칸 금지,
  논리/플랫폼 2레이어, 드래그 배치/이동(스왑)/회전/회수/투명화, 블랙홀-화이트홀 쌍.
- **웨이브**: 인간이 상단에서 낙하 스폰, 웨이브가 오를수록 강해짐. 자동 웨이브 토글.
- **전투·전염**: 근접 전투, 인간을 잡으면 그 자리에서 노예로 전염. 처치 경험치 → 레벨업.
- **캐릭터 정보 패널**: 캐릭터 탭 → 레벨/직업(미분류)/경험치/공격력/스킬트리(자리).
- **저장**: localStorage 자동 저장/복원. 프리스티지 상태 훅 포함(향후 구현).

로드맵·설계 배경은 [PLAN.md](./PLAN.md) 참고 (직업 분류, 스킬트리, 추적 이동 AI, 프리스티지).
